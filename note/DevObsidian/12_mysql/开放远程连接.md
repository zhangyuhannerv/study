## 5.7
先连接到本地数据库

切换到mysql数据库
```sql
use mysql
```

使用以下命令可以更改远程连接的设置
```sql
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'root' WITH GRANT OPTION;
```

刷新权限
```sql
flush privileges;
```

查询user表看看是否生效，如果 '%' 'root' 在第一行证明生效了
```sql
select host,user from user;
```
